## About

Text based social media app with gym theme, allows you to join different communities and add nested comments to posts.


## Tech Used

 - TypeScript
 - NextJS 14
 - Prisma
 - PostgreSQL
 - Clerk (auth)
 - UploadThing (profile photo uploading)
 - WebHooks (to keep clerk and db in sync)
 - Vercel (deployment)
